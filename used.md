Конфиг инжектов представляет собой json массив. каждое правило описано деревом webinjects. элементы дерева содержат правила для замены фрагментов кода.

Поле host - хост инжектируемого узла. поддерживает маску *
Поле path - path узла. поддерживает маску *
На одном узле могут быть несколько инжектов. В этом случае инжекты идут через запятую.
Конфиг  узлов тоже идут через запятую.

Символы "" в теле инжектов должны быть экранированы символом \. например <script id=\"webinject_id\"></script>

{
   "webinjects": 
   [
      {
         "host": "*yandex.ru*",
         "path": "*",
         "data":
         [
            {
               "inject": "</title>",
               "before": "",
               "after": ""  
            },
	    {
               "inject": "HELLO",
               "before": "",
               "after": "</head>"  
            }
         ]
      },
	  {
         "host": "*mail.ru*",
         "path": "*",
         "data":
         [
            {
               "inject": "</title>",
               "before": "",
               "after": ""  
            },
	    {
               "inject": "<script>alert('hello world');</script>",
               "before": "",
               "after": "</head>"  
            }
         ]
      }
   ]
}
